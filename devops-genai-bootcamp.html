DevOps with GenAI BootCamp

In this course we will learn how to build, containerize, deploy, and automate a GenAI-powered chatbot using Docker, Kubernetes, and GitHub Actions. Learn how to build, containerize, deploy, and automate a GenAI-powered chatbot using Docker, Kubernetes, and GitHub Actions.

Syllabus

Workshop Content

DEVOPS WITH GENAI – DOCKER, KUBERNETES & CI/CD IN ACTION

  • What is GenAI? (Beginner-friendly explanation)
  • How GenAI applications work (Prompt → Inference → Response)
  • Understanding AI Chatbot architecture (Frontend, Backend, LLM API)
  • Overview of Large Language Models (LLMs)
  • CPU vs GPU usage in GenAI applications (high level)
  • Docker for GenAI Applications
    • Containerizing an AI chatbot application
    • Writing a Dockerfile for a Python Flask app
    • Running chatbot in Docker (Free mode vs API mode)
    • Docker image build and push to DockerHub
  • Kubernetes for GenAI Applications
    • Deploying chatbot on Kubernetes
    • Understanding Pod, Deployment, and Service
    • Scaling chatbot replicas
    • Managing environment variables and secrets
  • CI/CD using GitHub Actions
    • What is CI/CD in real-world projects?
    • Automated Docker image build using GitHub Actions
    • Pushing images to DockerHub
    • Deploying application to Kubernetes using CI/CD
  • End-to-end DevOps demo: Code → Docker → Kubernetes → CI/CD
  • Best practices for deploying GenAI applications safely
